Sipeed Maixduino Kit RISC-V

3000.00 EGP

The Sipeed Maixduino Kit is a development board based on the RISC-V architecture, featuring the K210 chip. It includes a 64-bit dual-core processor, 8MB of SRAM, and 16MB of flash memory. The kit supports various AI and machine learning applications with built-in support for TensorFlow Lite. It has GPIO pins, I2C, SPI, and UART interfaces, making it ideal for IoT projects, robotics, and AI applications. The Maixduino is compatible with the MaixPy Python environment, offering easy programming for developers.
